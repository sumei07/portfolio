# Personal Portfolio Website
This is my personal portfolio website, built as part of my journey in learning web development.
It showcases my skills, projects, and progress as I continue to grow as a developer.
This project currently serves as a blueprint for my final portfolio, and I am actively working on improving its design, functionality, and content.

# Technologies Used
- HTML
- CSS
- Bootstrap
- JavaScript

# Features
- Responsive navigation bar
- Hero section with introduction
- About Me section
- Skills showcase
- Project display using cards
- Contact form with basic interaction
Custom styling and hover effects
